# hubot-lgtm

A hubot script for LGTM.

See [`src/lgtm.coffee`](src/lgtm.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-lgtm --save`

Then add **hubot-lgtm** to your `external-scripts.json`:

```json
[
  "hubot-lgtm"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
